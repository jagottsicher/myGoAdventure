# myGoAdventure
## Recoding Atari2600 "Adventure"-like game for the terminal with Go

Similar like the original, but different in many ways. Maybe better? 

Done for educational purposes, as a proof of principle, to become fimiliar with packages tcell, and last but not least to get to the bottom of some Go programming concepts.

Dedicated to Warren Robinett's great game Adeventure from 1 which provided me seemingly endless hours of thrill, frustration and joy!
